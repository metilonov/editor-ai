# Архитектура

Поток: Telegram/URL → загрузка → ffprobe → анализ видео/аудио → кандидаты → scoring/diversity → субтитры → рендер профиля → manifest/thumbnail → отправка.

Анализаторы независимы и возвращают нормализованные временные ряды. Scoring объединяет их весами профиля. Renderer строит один FFmpeg filtergraph и не перекодирует промежуточные клипы несколько раз.
