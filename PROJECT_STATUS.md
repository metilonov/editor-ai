# Состояние проекта

Это расширенная рабочая версия, а не набор пустых папок. Ключевые алгоритмы, рендер, база, очередь, Telegram-интерфейс, CLI и тесты реализованы.

Не заявляются как реализованные: человеческое понимание сюжета без нейросети, бесплатный безлимитный STT, одобренная TikTok Content Posting API и круглосуточный запуск без среды выполнения Python.
